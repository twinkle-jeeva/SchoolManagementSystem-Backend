# School Management System (Backend)

Partial backend for a multi-role school management system built with **.NET 10**, **EF Core**, and **C#**.  
Currently **under development** as part of my Full-Stack learning journey.

---

## Tech Stack
- C# .NET 10
- EF Core
- SQL Server
- REST APIs
- JWT Auth (planned)
- Swagger/OpenAPI

---

## Features (In Progress)
- CRUD for **Student**
- Models: StudentProfile, Club, Course
- Multi-role system in progress
- Swagger UI enabled
